# 🚗 StyleCar - 자동차 용품 전문 쇼핑몰

Next.js 기반의 완전한 이커머스 웹사이트

## ✨ 주요 기능
- 🛍️ 상품 카탈로그 및 검색
- 🛒 장바구니 및 주문 시스템
- 👤 회원가입 및 로그인
- 📱 반응형 디자인
- ⚡ 빠른 성능 최적화

## 🛠️ 기술 스택
- **Frontend**: Next.js 14, React, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Deployment**: Vercel
- **Domain**: stylecar.co.kr

## 🚀 배포
이 프로젝트는 Vercel을 통해 자동 배포됩니다.
